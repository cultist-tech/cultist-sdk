Админ:
- создает нфт
- указывает что нфт для реферальной системы
- указывает адрес инфлюенсера для этой нфт
- добавляет в пулл нфт для реффералки

Юзер:
- минтит случайную реферальную нфт из пулла
- юзер привязывается к инфлюенсеру, и инфлюенсеру приписывается роялти от продаж юзера

----

Админ:
- генерит код
- дает ссылку инфлюенсеру

Юзер:
- проходит по ссылке
- если подключен кошелек, автоматически привязывается к инфлюенсеру и приписывается роялти

----

Реферальная платформа:
- создается проект
- добавляются адреса контрактов которые будут поддерживать реферальную систему
- в каждый контракт интегрируется код для поддержки рефералки

Админ:
- генерит код
- дает ссылку инфлюенсеру

Юзер:
- проходит по ссылке
- если подключен кошелек, идет запрос в систему и происходит привязка ко всем реферальным проектам (к примеру: 2 нфт контракта и маркет, на каждом контракте произойдет привязка)
